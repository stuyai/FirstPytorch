# Pytorch First Lesson

## Getting Started

Type this into the command line:
```shell
git clone [git@github.com:stuyai/FirstPytorch.git](https://github.com/stuyai/FirstPytorch.git)
cd FirstPytorch
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
